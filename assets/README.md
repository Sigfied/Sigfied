# Asset Usage

`profile-banner.svg` is the profile hero banner for the GitHub README.

## Use

- Embed it with a relative path from the repository README, for example: `![Profile banner](./assets/profile-banner.svg)`.
- Keep the banner text unchanged unless the approved profile copy changes.
- The SVG is intentionally simple, so it should stay easy to maintain and update.

## Design Notes

- Deep navy background with restrained cyan and blue accents.
- Built for readability first, with minimal decoration and no noisy effects.
- Sized for a GitHub profile header layout.
